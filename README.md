# TaoTrails

TaoTrails is a static MVP for an English Taoist culture and sacred China travel site.

The first version is intentionally small:

- A culture hub for Taoism basics.
- A sacred mountains cluster for travel-intent readers.
- Two interactive retention tools: a temple etiquette checklist and a tea brewing timer.
- Cross-links to the existing China travel planner series through ChinaPass.ai style CTAs.
- SEO basics: robots, sitemap, static pages, and a lightweight local checker.

## Local Preview

```bash
npm run dev
```

Then open `http://localhost:4321`.

## Validation

```bash
npm run check
```

## Deployment

This site has no build step. Deploy the repository root to Cloudflare Pages or any static host.
